### annotaterb にパッチを投げた
下記 3 つのパッチを投げました。
- [Migrate issue templates to folder\-based format\#320](https://github.com/drwl/annotaterb/pull/320)
- [Add unit tests for AnnotateRb::Helper\#321](https://github.com/drwl/annotaterb/pull/321)
- [Support opt\-in display of PostgreSQL enum types in annotations\#324](https://github.com/drwl/annotaterb/pull/324)

PostgreSQL の enum type を annotation に出力するというのは、割と前からやってみたかったのでやっとできてよかったな感はあります。<br>
ただ、`connection.enum_types`という API が Rails7.0 以前にはないこと、また version ごとに返す型が少し違ったりしていたので、そのあたりの考慮は必要でしたが、思ったよりサクッとできて👍<br>

他は Issue template が機能してなかったりの修正をしたり、テストを追加したり、、をしてました。<br>
Rubocop の Issue template がかなりリッチになっていて、こんなことできるの..?と驚いていて、かなり参考にさせてもらいました。
